Steps to prepare files for upload

Generalguidelines

  1. Column names are NOT case sensitive
  2. Column number and sequence MUST be exacetlly like example below
  3. Files with 5,000 or more rows must be saved as CSV format, smaller files are also suggest to save in CSV format
  4. Suggest to compress files to ZIP format before uploading to save upload time
  5. When changing and saving files, make sure that the leading zeros in license numbers are not removed by Excel.
  6. It's not necessary to remove the records without a license number.
